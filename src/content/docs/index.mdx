Atlas is a **critical-minerals derivatives and reference-price platform**, built around
four product surfaces that share one codebase family. The stack is deliberately small:
a Python FastAPI application per environment, a separate oracle capture process, and
PostgreSQL. Changes are developed locally, soaked on a staging environment that mirrors
production, and only then promoted through a controlled, manually-gated path.

## The four surfaces

<CardGrid>
  <Card title="Perpetuals exchange" icon="bars">
    A central-limit-order-book perpetual-futures venue: 10 markets across 7 metals
    (lithium, cobalt, nickel, copper, aluminum, zinc, silver) plus 3 composite indices.
    dYdX-v4-shaped REST + WebSocket, a full price-time-priority matching engine,
    funding, liquidations, and margin/risk controls.
  </Card>
  <Card title="Reference-price oracle" icon="magnifier">
    A multi-tier, multi-source price oracle blended by liquidity-weighted median into
    per-mineral reference rates and index levels, plus the perp-implied *Atlas Spot*
    engine. Published as a licensed feed — public view time-delayed, licensees real-time.
  </Card>
  <Card title="Market-making book" icon="star">
    An automated market-making book that quotes every Atlas market from one engine,
    co-located in-process with the matching engine. Its economics are the engine's
    spread capture net of adverse selection, measured today against simulated flow in
    the test environment.
  </Card>
  <Card title="Atlas Swaps" icon="random">
    A block / RFQ swaps workflow for institutional flow, consuming the oracle as its
    pricing reference.
  </Card>
</CardGrid>
